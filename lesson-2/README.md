ключевое слово const создает константу
константа требует обязательного указания типа
константа не может быть переопределена или быть создана другая константа с таким же именем

кортеж создается через ключевое слово tuple
в типе данных в круглых скобках перечисляются типы данных элементов кортежа, соответственно задается количество элементов
обращение к элементу кортежа через точку и последующее указание индекса элемента в кортеже. Например: .0

Если кортеж отметить как let mut, то значения элементов кортежа можно изменять без нарушения типизации

Массив создается через []
В типе данных в квадратных скобках указывается тип элементов и их количество
обращение к элементам чрез указание индекса в квадратных скобках
