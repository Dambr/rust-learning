cargo new - создать новый проект
cargo build - собрать проект
cargo run - запустить проект
cargo check - проверить проект на наличие ошибок компиляции
cargo clean - удалить папку target
rustfmt <путь к файлу> - выполнить форматирование указанного файла
собранный проект - в target/debug/<имя проекта> 

let - оператор создания переменной
let mut - оператор создания переменной, которую сожно изменить

Если попытаться присвоить в mut переменную значение другого типа, то будет ошибка

Можно повторно объявить переменную с тем же именем через оператор let. В этом случае прежняя переменная будет удалена и при попытке инициализации ее значением другого типа ошибки не будет

При создании новой переменной с ранее используемым имененм можно использовать значение старой переменной для формирования нового

Допустимые типы данных для работы с целыми числами:
i8, i16, i32, i64, i128
u8, u16, u32, u64, u128

Допустимые типы данных для работы с числами с плавающей запятой:
f16, f32, f64, f128

Логический тип данных bool

Тип данных для символов: char
